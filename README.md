# AFK_801
### TFM活动挂机

* 自动执行的操作：  
       1. 声音提醒  
       2. 弹出窗口（在全屏游戏/电影上强制弹出TFM窗口）  
       3. 自动按键（防挂机死等）  
       4. ~~？全自动？~~  
       5. .ahk文件可自行设置功能（在最底下）  
*  食用方法：  
       1.安装软件[AutoHotkey](https://github.com/AutoHotkey/AutoHotkey/releases)(.exe文件）  
       2.下载对应[活动脚本](https://github.com/lyliny/AFK_801/releases)（.ahk文件）  
       3.双击下载的脚本即可食用
